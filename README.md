ShopBridge is a role-based web application that connects shop owners and buyers.
Buyers can browse products, while shop owners can manage product listings.

## Features
- Role-based access (Buyer / Shop Owner)
- Shop-wise product browsing
- Product management for shop owners
- Simple and user-friendly interface

## Tech Stack
- Python (Flask)
- HTML
- CSS
- JavaScript

## How to Run Locally
1. Install Python
2. Install Flask:
   pip install flask
3. Run the application:
   python app.py
4. Open browser and go to:
   http://127.0.0.1:5000
